Sorting Algorithms Information

  • Selection Sort

    How it works: Selection sort repeatedly finds the minimum element from the unsorted part and moves it to the beginning.

    Markers: Orange highlights the current minimum, red indicates the element being compared, blue shows elements being swapped.

  • Insertion Sort

    How it works: Insertion sort builds the final sorted array one item at a time by inserting elements into their correct position.

    Markers: Orange highlights the current key element, red indicates the element being compared.

  • Bubble Sort

    How it works: Bubble sort repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.

    Markers: Red indicates elements being compared, blue shows elements being swapped.

  • Shell Sort

    How it works: Shell sort sorts elements that are far apart and successively reduces the interval between elements to be sorted.

    Markers: Orange highlights the current element being inserted, red indicates elements being compared.

  • Merge Sort

    How it works: Merge sort divides the array into halves, sorts them recursively, and then merges them.

    Markers: Orange indicates current elements being merged.

  • Quick Sort

    How it works: Quick sort divides the array into sub-arrays based on a pivot and recursively sorts them.

    Markers: Purple highlights the pivot, red indicates elements being compared, blue shows elements being swapped.

  • Heap Sort

    How it works: Heap sort converts the array into a heap data structure and repeatedly extracts the maximum element to sort the array.

    Markers: Blue shows elements being swapped during heapify and extraction.
